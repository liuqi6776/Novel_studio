# 伏笔追踪表

## 使用说明

- **编号**: F001, F002, F003... 按埋下顺序编号
- **类型**: 物品 / 秘密 / 人物 / 事件 / 规则 / 关系
- **状态**: 埋下 / 推进中 / 已回收 / 已废弃
- **回收章节**: 若暂未确定，可先标“待定”，但必须写回收计划

## 追踪表

| 编号 | 内容描述 | 埋下章节 | 类型 | 状态 | 回收章节 | 回收触发 / 计划 | 备注 |
| ---- | -------- | -------- | ---- | ---- | -------- | --------------- | ---- |
| F001 |          | Ch.XXX   |      | 埋下 | 待定     |                 |      |

## 检查清单

每写完 10 章检查：

- [ ] 新增伏笔是否已记录
- [ ] 是否有伏笔可进入回收阶段
- [ ] 伏笔编号是否连续
- [ ] 伏笔状态是否与章节实际进度一致

每写完 1 卷检查：

- [ ] 本卷伏笔回收情况
- [ ] 需要延续到下卷的伏笔
- [ ] 需要废弃或改写的伏笔
