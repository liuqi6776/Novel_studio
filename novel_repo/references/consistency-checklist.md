# 一致性检查清单

## 输出格式

每次检查至少输出：问题点 / 证据段落 / 修正建议。

若问题较多，建议加一列严重度：

- 🔴 硬冲突：事实矛盾，必须修复
- 🟡 软冲突：逻辑不严密，建议修复
- 🟢 风格偏差：不影响剧情，可选修复

## 每 10 章检查

### 剧情一致性

- [ ] 是否有前后矛盾的情节
- [ ] 时间线是否连贯
- [ ] 地点转换是否合理
- [ ] 事件因果关系是否清晰
- [ ] 重大真相是否遵守“线索 / 误判 / 校正”分层推进

### 人物一致性

- [ ] 角色行为是否符合性格与行为模式
- [ ] 角色能力是否超出世界观上限
- [ ] 角色关系是否双向一致
- [ ] 角色声线是否稳定
- [ ] 主角是否长期脱离固定配角互动

### 设定一致性

- [ ] 世界规则是否被遵守
- [ ] 每条规则的限制与代价是否保持一致
- [ ] 专有名词是否统一
- [ ] 距离与时间是否匹配

### 伏笔与同步

- [ ] 新增伏笔是否已记录
- [ ] 是否有伏笔应进入回收阶段
- [ ] 角色状态变化是否已回写档案
- [ ] 阶段变化是否需要回写总览或卷纲

## 每卷检查

### 整体审查

- [ ] 卷主题是否明确
- [ ] 阶段目标是否完成
- [ ] 角色成长与关系变化是否成立
- [ ] 卷高潮是否包含事件、代价与未解问题

### 漏洞填补

- [ ] 剧情漏洞
- [ ] 设定漏洞
- [ ] 逻辑漏洞
- [ ] 时间线漏洞

### 伏笔清理

- [ ] 本卷伏笔回收情况
- [ ] 需要延续到下卷的伏笔
- [ ] 需要废弃或改写的伏笔

## 完稿检查

### 全局一致性

- [ ] 主线剧情完整
- [ ] 所有主要伏笔完成闭环或有明确去向
- [ ] 角色结局与弧线匹配
- [ ] 世界观自洽

### 细节统一

- [ ] 人名 / 地名统一
- [ ] 时间线完整
- [ ] 力量体系一致
- [ ] 术语统一

### 质量检查

- [ ] 无明显错别字
- [ ] 段落过渡流畅
- [ ] 对话自然
- [ ] 描写生动
