# 项目写作规则

## 风格定位

### 核心风格
**欢快吐槽流**——参考《不科学的御兽》的轻松节奏 + 《大王饶命》的吐槽风格。

### 具体规则

1. **主角视角为主**：以主角林渊的第一人称视角或紧密第三人称视角叙述。主角的内心吐槽是本书的核心魅力。

2. **游戏化思维贯穿**：主角是游戏策划穿越，遇到任何事物都会用游戏术语类比。这是本书最大的特色。
   - 虚渊 = 卡池
   - 灵 = 角色卡
   - 契约 = 抽卡
   - 品阶 = R/SR/SSR/UR
   - 战斗 = 配对，正常对战，无回合制

3. **对话要有趣**：
   - 主角和小九的互怼是日常笑点
   - 王铁柱的憨厚吐槽是调味剂
   - 苏晚晴的高冷+偶尔破防是反差萌

4. **节奏要快**：
   - 每章必须有推进（剧情/认知/关系至少推进一项）
   - 每章结尾必须有钩子
   - 不要大段说明文，世界观通过对话和行动展示

5. **情绪要有起伏**：
   - 好笑→揪心→兴奋→解气→温暖→期待
   - 不能一直搞笑，也不能一直压抑
   - 打脸要有铺垫，不能太容易

6. **战斗描写**：
   - 灵的战斗要有策略性，不是单纯对轰
   - 利用灵的种族克制、技能配合、弱点攻击
   - 后期多灵配队是核心看点

## 设定规则

### 灵位系统
| 境界 | 灵位数量 | 可契约灵 |
|------|----------|----------|
| 感灵境 | 1个 | 凡灵 |
| 驭灵境 | 2个 | 凡灵、异灵 |
| 化灵境 | 3个 | 凡灵、异灵、玄灵 |
| 灵王境 | 4个 | 凡灵~地灵（4个是上限） |
| 灵皇境 | 4个 + 唯一灵 | 可以感应并契约唯一灵 |
| 灵帝境 | 4个 + 唯一灵 | 唯一灵能力完全解放 |

### 灵的品阶
| 品阶 | 名称 | 稀有度标签 |
|------|------|------------|
| 1阶 | 凡灵 | R |
| 2阶 | 异灵 | SR |
| 3阶 | 玄灵 | SSR |
| 4阶 | 地灵 | UR |
| 5阶 | 天灵 | LR |
| 6阶 | 唯一灵 | UNIQUE |

### 灵的种族
- 龙裔族：龙类灵，强大稀有
- 风翎族：飞行类灵，速度快
- 灵植族：植物类灵，控制/治疗
- 岩魄族：岩石类灵，防御高
- 幽魂族：暗影类灵魂体，隐匿/诡异
- 幻妖族：精神类灵，幻术/控制
- 古神族：唯一灵专属种族，世界规则的体现

## 长线连载风险控制

### 1. 吐槽风的"边际递减效应"规避
- **问题**：前100章用"游戏抽卡、非酋、掉率"吐槽很新鲜，但过了100章如果还在用，会削弱事件严肃性
- **解决方案**：
  - 第二卷后段或第三卷时，安排一次"破局"事件
  - 让主角意识到"灵不只是数据，而是有血有肉的伙伴"
  - 完成一次心智成长，对应"认知变化点"和"不可逆损失"
  - 转折后：吐槽可以保留，但底层心态要转变——从"玩游戏"到"背负生命"

### 2. 灵视金手指的审美疲劳规避
- **问题**：如果每次收服新灵都是【开灵视→发现隐藏款→契约→打脸】，套路重复三次读者就腻了
- **解决方案**：
  - 进化路线要"严苛且奇葩"
  - 残灵变强不仅靠"发现隐藏款"，更要靠**材料收集**
  - 例如：需要吸收特定强敌的本源、需要在特定天象下突破、需要完成搞笑又艰难的任务
  - **爽点从"结果"转移到"过程"**
  - 每一次进化都是一场冒险，而不是简单的"看到了就拿到了"

### 3. 反派智商保鲜
- **问题**：如果反派只靠等级压制，毫无策略，战斗就变成无聊的对轰
- **解决方案**：
  - 赵无极等主要反派要有**配队策略**
  - 他们的灵组合要有"毒瘤卡组"式的恶心打法
  - 主角后期要靠"战术拆解"来取胜，而不是纯粹堆数值
  - 核心爽点：**斗法 → 智斗与博弈**

### 4. 不可逆损失机制
- 每卷至少安排1次"不可逆损失"
- 可以是：伙伴受伤/离去、错过机缘、留下永久伤痕
- 作用：让读者感受到这个世界有重量，主角的每一步选择都有代价

## 章节结构规则

每章必须包含：
1. **开场锚点**：时间/地点/视角/当前状态
2. **核心冲突**：本章要解决的核心问题
3. **剧情节点**：5个节点（开场→发展→转折→高潮→收束）
4. **钩子**：章末悬念
5. **记忆更新清单**：本章新增/变化的设定

## 禁忌

- ❌ 不要大段说明文（世界观通过对话和行动展示）
- ❌ 不要让主角太顺利（打脸要有代价）
- ❌ 不要让灵变成工具（每个灵都要有性格）
- ❌ 不要忘记伏笔（每章检查伏笔追踪表）
- ❌ 不要让战斗变成对轰（要有策略和配合）
- ❌ 不要压抑太久（爽点要及时释放）
