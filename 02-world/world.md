# 《万灵契约》世界观设定

## 核心命题

这个世界没有灵气，只有"灵"。灵来自名为"虚渊"的异空间。人类通过感应虚渊召唤灵，与灵签订魂契获得力量。

**核心冲突**：有限的灵 vs 无限的欲望。尤其是唯一灵——天地间只有一只，全大陆的强者都在抢。

---

## 世界规则

### 规则一：灵位限制

- **规则**：人类灵魂中容纳灵的位置称为"灵位"，数量随境界提升
- **限制**：
  - 感灵境：1个灵位
  - 驭灵境：2个灵位
  - 化灵境：3个灵位
  - 灵王境：4个灵位（普通灵上限）
  - 灵皇境及以上：4个普通灵位 + 1个唯一灵位
- **代价**：灵位有主后强行移除，灵魂会受到不可逆损伤
- **破坏后果**：4个普通灵位全空者，灵魂崩溃，必死无疑

### 规则二：魂契绑定

- **规则**：人与灵通过"魂契"链接，契约后灵与主人生命相连
- **限制**：
  - 契约需要双方同意（强制契约成功率极低且反噬严重）
  - 契约后灵无法离开主人超过百丈
  - 主人死亡，灵回归虚渊；灵死亡，主人灵魂受创
- **代价**：契约过程消耗灵魂力，越强的灵消耗越大
- **破坏后果**：强行撕毁契约，双方重伤，严重者灵魂破碎

### 规则三：换灵代价

- **规则**：可以解除旧灵契约，契约新灵
- **限制**：
  - 每次换灵需要"断契仪式"，过程极其痛苦
  - 换灵后灵魂留下"裂痕"，需要长时间修复
  - 短时间内多次换灵，裂痕叠加，可能导致灵魂崩溃
- **代价**：
  - 1次换灵：虚弱1个月，灵魂力下降30%
  - 2次换灵（短时间内）：虚弱3个月，灵魂力下降50%
  - 3次换灵（短时间内）：九死一生
  - 4灵全换：必死无疑
- **破坏后果**：无（这是规则本身）

### 规则四：唯一性法则

- **规则**：某些强大的灵天地只生一只，称为"唯一灵"
- **限制**：
  - 唯一灵无法被复制或量产
  - 唯一灵死亡后，虚渊会在随机时间、随机地点重生该灵
  - 唯一灵契约难度极高，需要特殊条件（如击败灵本身、满足灵的执念等）
  - 只有灵皇境以上才能感应到唯一灵的存在
- **代价**：唯一灵的力量伴随诅咒或限制，越强代价越大
- **破坏后果**：唯一灵若被恶意毁灭，可能引发虚渊震荡

### 规则五：虚渊感应

- **规则**：人类通过"感应"与虚渊建立联系，召唤灵前来契约
- **限制**：
  - 每次感应有时间限制（通常1个时辰）
  - 感应范围受境界限制，境界越高能感应到的灵越强
  - 感应到的灵是随机的——说白了就是**抽卡**
  - 低境界时能感应到的灵很少，大部分都模糊不清
- **代价**：感应消耗精神力，过度感应会导致精神崩溃
- **破坏后果**：强行突破感应限制，可能引来虚渊深处的"东西"

## 灵的体系

### 灵的品阶

| 品阶 | 名称 | 稀有度 | 说明 |
|------|------|--------|------|
| 1阶 | 凡灵 | R | 最常见的灵，能力单一，如"灰毛鼠""青藤" |
| 2阶 | 异灵 | SR | 有特殊能力的灵，如"雷鹰""毒龙草" |
| 3阶 | 玄灵 | SSR | 拥有智慧的灵，可自主战斗，如"冰鸾""九尾狐" |
| 4阶 | 地灵 | UR | 一方之霸，能力强大，如"山岭巨人""炎龙" |
| 5阶 | 天灵 | LR | 传说中的灵，极其稀有，如"时间之蛇" |
| 6阶 | 唯一灵 | UNIQUE | 天地独一份，能力逆天，如"命运之轮""虚空之眼" |

### 灵的种族

#### 1. 岩魄族（坦克/前排）
- **特点**：防御高，血厚，擅长控制和守护
- **代表灵**：石心守卫（凡灵）、玄铁傀儡（异灵）、山岭巨人（地灵）
- **适合位置**：前排肉盾

#### 2. 风翎族（刺客/输出）
- **特点**：速度快，敏捷高，擅长突袭和连击
- **代表灵**：疾风隼（凡灵）、影豹（异灵）、雷鹰（玄灵）
- **适合位置**：物理输出

#### 3. 幽魂族（控制/特殊）
- **特点**：诡异莫测，擅长诅咒、附身、灵魂攻击
- **代表灵**：噬魂鬼（异灵）、梦魇（玄灵）、幽灵骑士（地灵）
- **适合位置**：控制、debuff

#### 4. 灵植族（辅助/治疗）
- **特点**：恢复能力强，擅长辅助、治疗、毒素
- **代表灵**：回春藤（凡灵）、毒龙草（异灵）、生命之树（地灵）
- **适合位置**：奶妈、辅助

#### 5. 龙裔族（核心/全能）
- **特点**：攻防兼备，稀有度高，成长上限高
- **代表灵**：炎龙幼崽（玄灵）、冰龙（地灵）、雷霆古龙（天灵）
- **适合位置**：核心输出/副T

#### 6. 幻妖族（控制/奇袭）
- **特点**：变化多端，擅长幻术、迷惑、偷袭
- **代表灵**：幻蝶（异灵）、镜妖（玄灵）、九尾狐（地灵）
- **适合位置**：控制、奇袭

#### 7. 古神族（唯一灵专属）
- **特点**：独一无二，能力逆天，但代价巨大
- **代表灵**：时间之蛇、命运之轮、虚空之眼、因果之线
- **适合位置**：王牌、底牌

### 灵技类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 攻击技 | 直接造成伤害 | 火焰吐息、雷霆一击 |
| 防御技 | 抵挡或减少伤害 | 石肤术、护盾 |
| 控制技 | 限制敌人行动 | 冰冻、眩晕、束缚 |
| 辅助技 | 增强己方 | 加速、治疗、强化 |
| 特殊技 | 独特效果 | 瞬移、隐身、复活 |

### 灵阵（多灵配合）

当多个灵同时出战时，可以激活"灵阵"效果：

- **双灵阵**：两个灵配合，产生1+1>2的效果
- **三灵阵**：三个灵配合，产生质变
- **四灵阵**：四个灵完美配合，威力惊人
- **唯一灵阵**：唯一灵 + 四灵配合，毁天灭地

**经典配队示例**：

| 配队名 | 阵容 | 效果 |
|--------|------|------|
| 铁壁流 | 岩魄+岩魄+灵植+灵植 | 极致防御+无限续航 |
| 速攻流 | 风翎+风翎+风翎+幽魂 | 先手秒杀，不给反应时间 |
| 龙威流 | 龙裔+龙裔+岩魄+灵植 | 双核输出+前排+奶妈 |
| 控制流 | 幽魂+幻妖+灵植+岩魄 | 控到死，慢慢磨 |
| 均衡流 | 岩魄+风翎+灵植+龙裔 | 万金油，什么都能打 |

## 地理与势力

### 青萝宗（主角起点）
- **等级**：二流宗门
- **位置**：青云山脉
- **特点**：以培养御灵师为主，外门弟子数千，竞争激烈
- **镇宗之灵**：4阶地灵"青萝仙子"（灵植族）
- **宗门特色**：每年举办"启灵大典"，外门弟子统一感应虚渊

### 其他势力（待展开）
- **天玄宗**：一流宗门，正道领袖，拥有2只唯一灵
- **幽冥教**：邪道，研究禁忌换灵术，试图量产唯一灵
- **万兽山**：以御兽为主，与御灵有相似之处，两派常年较劲
- **散修联盟**：自由御灵师组织，信息集散地

## 社会体系

### 权力结构
1. **宗门高层**：宗主、长老，掌握稀有灵的信息和资源
2. **核心弟子**：内门精英，拥有优质灵，享受最好资源
3. **普通弟子**：外门弟子，资源有限，竞争激烈
4. **散修**：无门无派，自由但缺乏保障
5. **凡人**：无法感应虚渊，只能依附御灵师

### 经济体系
- **货币**：灵石（蕴含灵魂力的石头）
- **核心资源**：
  - **灵**：最核心资源，可交易但风险高（契约后的灵无法转让）
  - **魂晶**：修复灵魂损伤的稀有材料，换灵者必备
  - **灵鉴**：记录灵信息的图鉴，稀有灵的信息价值连城
  - **功法**：提升灵魂力的修炼法门
- **灵市**：专门交易灵的地方（未契约的灵）

### 禁忌
- 禁止强行契约他人已契约的灵
- 禁止大规模屠杀灵（会引发虚渊震荡）
- 禁止研究"夺舍"（占据他人身体获取灵位）
- 惩罚：轻则废除修为，重则处死

## 文化与习俗

### 启灵大典
每年一次，年满16岁的外门弟子统一感应虚渊，尝试契约第一灵。这是每个御灵师最重要的时刻——**第一次抽卡**。

### 契约礼
成功契约灵后，举行小型庆祝。但如果契约的是"废灵"……庆祝就变成了嘲笑大会。

### 换灵祭
不得已换灵时，祭拜虚渊，祈求平安。因为换灵是真的会死人的。

## 历史与秘密

### 已知历史
- 千年前，虚渊突然出现，人类开始能感应灵
- 五百年前，第一位灵帝境出现，契约了第一只唯一灵
- 百年前，虚渊出现异动，灵的出现频率增加
- 十年前，虚渊裂缝扩大，有"虚兽"逃出

### 隐藏秘密（逐步揭晓）
1. 虚渊不是自然形成，而是某种"造物"
2. 灵可能是某种高级存在的"投影"或"碎片"
3. 人类与灵的契约，本质是一种"交易"
4. 主角的双魂，可能与虚渊的创造者有关
5. 唯一灵的总数是固定的——108只

## 可验证场景

1. **抽卡场景**：感应虚渊必须体现随机性和紧张感——"这次能出什么？"
2. **契约场景**：契约过程必须体现灵魂力的消耗和契约的绑定感
3. **战斗场景**：战斗必须体现灵的配合和灵阵的效果，不是单挑是团战
4. **换灵场景**：换灵必须体现痛苦和灵魂损伤
5. **唯一灵场景**：唯一灵的出现必须体现其稀有和强大——"全世界只有这一只"
