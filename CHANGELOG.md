# Changelog

All notable changes to novel-master should be documented in this file.

## [1.3.0] - 2026-03-07

### Added

- README，补充项目定位、快速开始、CLI 说明、适用边界与开源说明
- CONTRIBUTING，补充贡献范围、规则要求、模板要求与 PR 建议格式
- examples/minimal-workspace，补充脱敏最小工作区示例
- COMPATIBILITY，补充主规范与 CLI / 模板资源的差异说明
- 开源首个正式发布版本

### Changed

- 收敛 assets/templates 的 overview、rules、角色、世界观、卷纲、章节与伏笔模板口径
- 收敛 references 的章节骨架、角色模板、世界观指南、文风指南与一致性检查清单
- 更新 CLI 的 create 流程，初始化时同步生成 rules、世界观、主角档案和伏笔追踪文件

### Notes

- 当前 CLI 仍定位为轻量脚手架，不应视为完整创作引擎
